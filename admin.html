Configuración de GitHub

Para usar el guardado automático, necesitas configurar tu token de GitHub:

1. Crear Personal Access Token

Ve a GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token

2. Permisos necesarios

Selecciona: repo (Full control of private repositories)

3. Configurar datos
